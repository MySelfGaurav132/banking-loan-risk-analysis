
# Banking & Loan Risk Analysis System

## Description
This project implements a SQL-based banking system to analyze loan repayment
behavior and classify loans into risk categories.

## Features
- Relational database design using MySQL
- Realistic customer, loan, and repayment data
- Loan risk classification using SQL CASE logic

## Loan Risk Classification
Loans are classified into three categories:
- LOW: No missed repayments
- MEDIUM: One missed repayment
- HIGH: Two or more missed repayments

## Technologies Used
- MySQL
